3) (PHP,Apache/Nginx)Заблокировать вход на сайт всем, кроме определенного IP адреса. В данном задании нужно описать свое решение.
Примечание:
--если вы знаете несколько вариантов решения, опишите все.

- в коде можно проверять $_SERVER['REMOTE_ADDR']
в htaccess директивы
Order Allow,Deny
Allow from 192.168.0.     # разрешить всем в локальной сети
Deny from 192.168.0.2    # запертить конкретному ip в локальной сети

аналогично для внешней сети

Allow from 95.80.64.     # разрешить всем из выбранной сети
Deny from 10.60.20.2    # запертить конкретному ip из внешней сети
